This project was done with the main focus on the Javascript aspect of it, hence I hope it is possible for you to ignore the simplistic UI implemented.


You can add a book with the "New book" button in this panel.
Every book starts with a 3 star rating, you must drag the book to another shelf to modify its rating. The higher the book is in the library, the better its rating.


To remove a book from The Library, you need to drag it to the 'Give away' icon next to the 'New book' button.
